# 💱 DevClub - Conversor de Moedas

Aplicação web simples para conversão entre diferentes moedas (Real, Dólar, Euro, Bitcoin e Libra), desenvolvida como projeto prático do curso **DevClub** (Trilha Full Stack Pró).

🔗 **Demo online:** [felipemessiasalves.github.io/devclub-conversor-moeda](https://felipemessiasalves.github.io/devclub-conversor-moeda/)

![Preview do projeto](https://felipemessiasalves.github.io/devclub-conversor-moeda/assets/logo.gif)

## 📋 Sobre o projeto

O Conversor de Moedas permite que o usuário selecione uma moeda de origem, uma moeda de destino, informe um valor e obtenha o resultado convertido de forma rápida e visual. O projeto foi construído aplicando os conceitos de **HTML**, **CSS** e **JavaScript** trabalhados no módulo de JavaScript Web do curso DevClub, ministrado pelo professor Rodolfo Mori.

## ✨ Funcionalidades

- Conversão entre as seguintes moedas:
  - 🇧🇷 Real Brasileiro (R$)
  - 🇺🇸 Dólar Americano (USD)
  - 🇪🇺 Euro (€)
  - ₿ Bitcoin (₿)
  - 🇬🇧 Libra Esterlina (£)
- Seleção independente da moeda de origem ("Converter de") e da moeda de destino ("Converter para")
- Campo de entrada para o valor a ser convertido
- Exibição do resultado com ícones representativos de cada moeda
- Interface simples, responsiva e de fácil utilização

## 🛠️ Tecnologias utilizadas

- **HTML5** – estruturação da página
- **CSS3** – estilização e responsividade
- **JavaScript** – lógica de conversão e interatividade

## 🚀 Como executar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/felipemessiasalves/devclub-conversor-moeda.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd devclub-conversor-moeda
   ```
3. Abra o arquivo `index.html` no navegador (ou utilize a extensão **Live Server** do VS Code para uma melhor experiência de desenvolvimento).

Não é necessária a instalação de dependências, pois o projeto é construído apenas com HTML, CSS e JavaScript puro (vanilla).

## 📁 Estrutura do projeto

```
devclub-conversor-moeda/
├── assets/          # Imagens e ícones (logo, moedas, seta, etc.)
├── index.html       # Estrutura principal da aplicação

```

> A estrutura acima pode variar de acordo com a organização real dos arquivos no repositório.
